# README

enter console
```
bin/rails console
```

start server
```
bin/rails server
```

navigate to e.g. http://localhost:3000/groups.json

see available middleware
```
bin/rails middleware
```

create db
```
bin/rails db:create
```

migrate
```
bin/rails db:migrate
```
